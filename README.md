# Cute Date Invitation Website 💖

A playful and romantic React web application designed as an interactive date invitation.  
The project includes a cute landing page, a moving “No” button, a date and time picker, a food preference selection page, and a final confirmation screen.

## Overview

This project is a fun single-page React application created to make asking someone out more personal and memorable.  
The user is first asked if they would like to go on a date. If they try to click “No”, the button moves away from the cursor, making it difficult to reject the invitation. When the user clicks “Yes”, the application continues through a sweet and interactive flow.

## Features

- Cute and romantic pink-themed design
- Responsive layout for desktop and mobile
- Interactive “No” button that moves away from the cursor
- Confirmation screen after selecting “Yes”
- Native date picker with past dates disabled
- Time selection dropdown
- Food preference selection page
- Final personalized date confirmation page
- Decorative flowers, hearts, paws, and puppy-inspired styling
- Built with reusable React components

## User Flow

1. The user sees the main question:  
   **“Will you go on a date with me?”**

2. The user can choose:
   - **Yes** — continues to the next page
   - **No** — the button moves away when the cursor gets close

3. After clicking **Yes**, a surprise message appears.

4. The user selects:
   - a date
   - a time

5. The user chooses what they would like to eat.

6. A final message confirms the plan in a cute and romantic way.

## Technologies Used

- React
- JavaScript
- CSS3
- Vite
